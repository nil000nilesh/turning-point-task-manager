App ka Parichay

Kya hai TPS Client Desk AI?

TPS Client Desk AI ek modern task management aur team coordination app hai jo specifically Tarningpoint Marketing ke liye banaya gaya hai. Yeh app Google login se secure hai aur real-time mein kaam karta hai — matlab ek member jo update kare woh turant baaki team ko dikh jata hai.

📋
Task Management
Tasks banana, assign karna, track karna. Leader kisi bhi member ko task de sakta hai. Member khud ke liye bhi task bana sakta hai.
🤖
AI Assistant (Hindi)
Hindi, English, Hinglish mein baat karo. AI khud samjhega aur task/reminder/note automatically banayega.
🔔
Live Notifications
Naya task mila, koi message aaya, reminder due hai — sab kuch instantly popup aata hai screen pe aur browser mein bhi.
📝
Notes Module
Personal notes likhne ke liye. Meetings, ideas, client details — sab notes mein save karo. Colors aur categories bhi set kar sakte ho.
💬
Task Conversations
Har task pe leader aur member chat kar sakte hain. Task kyu nahi hua, kaise karega — sab discuss kar sakte hain.
📊
Progress Tracking
Pending, In Progress, Done — task ka status dekhna aur update karna. Overdue tasks pe automatic alert aata hai.
💡 Tip: Yeh app mobile, tablet aur PC — teeno pe equally achha kaam karta hai. Mobile pe hamburger (☰) button se sidebar khulta hai.

3 🔐

Login karna

Pehli baar aur roz roz login kaise karein

Pehli Baar Login (New Member)

  • 1
    App kholo Browser mein app ka link kholo (Admin ne share kiya hoga)
  • 2
    "Google se Login Karo" pe click karo Apna Tarningpoint Google account select karo (ya jo email admin ne add ki ho)
  • 3
    PIN Setup karo (pehli baar) 4-digit security PIN choose karo. Yeh PIN roz login ke waqt maanga jayega. Yaad rakhna zaroori hai!
  • 4
    PIN Enter karo PIN confirm karo aur "Verify" karo. App open ho jayega.
ℹ️ Roz Login ke liye: Google login → PIN enter karo → App directly open hoga. PIN 30 din tak valid hai.
⚠️ PIN bhool gaye? Admin se contact karo. Admin aapka PIN reset kar dega, phir aap naya PIN set kar sakte ho.

Notifications Allow Karo (Important!)

Login ke baad browser notification permission maangega. "Allow" zaroor karo — warna naye tasks, messages aur reminders ka notification nahi aayega.

💡 Mobile pe: Settings → App Permissions → Browser → Notifications → Allow karo.

AI Chatbot — Complete Guide

Hindi mein baat karo, AI khud kaam karega

Screen ke niche-right corner mein 🤖 (robot) button hoga. Isse click karo → AI chat window khulega. Aap Hindi, English, ya Hinglish mein seedha likh sakte ho.

Quick Action Chips (Sabse Aasaan Tarika)

Jab bhi AI chat khulega, sabse niche 4 colored buttons (chips) dikhenge:

📋 Task 🔔 Reminder 📝 Note 📊 Status

Chip Use Karne ka Tarika (Step by Step)

📋 Task Chip:
  1. 📋 Task chip pe click karo
  2. AI poochega: "Kiske liye task banana hai?"
  3. Buttons dikhenge: 👤 Khud ke liye, [Member ke naam]
  4. Member select karo
  5. Input box mein task title likho → Enter dabao
  6. Task automatically ban jayega aur member ko notification jayega!
🔔 Reminder Chip:
  1. 🔔 Reminder chip pe click karo
  2. AI poochega: "Kiske liye reminder set karna hai?"
  3. Buttons: 👤 Khud ke liye, [Members], 👥 Sabko
  4. Select karo, phir reminder title likho → Enter
  5. Reminder save ho jayega!
📝 Note Chip:
  1. 📝 Note chip pe click karo
  2. Input box activate ho jayega
  3. Note content seedha type karo → Enter
  4. Note automatically "My Notes" mein save ho jayega
📊 Status Chip:
  1. 📊 Status chip pe click karo
  2. AI turant poori team ki status report dikhayega
  3. Koi extra kuch type karne ki zaroorat nahi

Task Conversation

Leader aur Member ke beech task pe chat

Har task pe leader aur member chat kar sakte hain. Isse clearly samajh aata hai ki task kyu nahi hua, kya problems hain, aur aage kaise hoga.

Task Chat Kaise Kholein

  • 1
    Tasks list mein apna task dhundhoYa "My Tasks" page pe jao
  • 2
    "💬 Chat" ya "Update" button dabaoTask chat window khulega
  • 3
    Message type karo → Send karoDusri party ko instantly notification aayega
💡 Best Practice:
  • Member: Task late ho rahi ho toh immediately chat karo — reason batao. Leader appreciate karega.
  • Leader: Agar member ne update kiya hai toh reply karo — guidance do. Isse team trust badhti hai.
  • Chat mein clear aur short messages likho — jaise WhatsApp pe karte ho.
Real Example:
Leader: "Rahul, website update abhi tak nahi hua. Kya problem hai?"
Rahul: "Sir, hosting pe issue tha. Aaj sham 6 baje tak complete kar dunga."
Leader: "OK, kal client ko demo dena hai. 6 baje pakka karni hai."

Tips, Best Practices & Common Mistakes

Yeh jaanna zaroori hai!

Best Practices — Leader ke liye

  • Roz subah status check karo — Dashboard → Overdue tasks pe sabse pehle dhyan do
  • Task assign karte waqt due date hamesha set karo — Bina due date ke member ko urgency nahi pata
  • Clear task title likho — "Website" ki jagah "Client ABC ki website homepage update karo"
  • Overdue task pe turant chat karo — Member se puchho kya problem hai, judge mat karo
  • Reminders use karo — Client follow-up, payment due dates, meetings — sab AI se set karo

Best Practices — Member ke liye

  • Task mili toh acknowledge karo — Task pe chat mein "Got it, kal tak karunga" likho
  • Status roz update karo — Pending se In Progress, phir Done. Leader ko pata chalega.
  • Late ho raho ho toh pehle batao — Deadline nikal jaane ke baad nahi, pehle leader ko chat pe batao
  • Notifications allow rakho — Warna naya task ya message miss ho sakta hai
  • Notes use karo — Client ke details, meeting notes — notes mein save karo, bhuloge nahi

Common Mistakes aur Solutions

Security — Zaroori Baatein

  • Apna 4-digit PIN kisi ko mat batao — yeh personal security hai
  • Admin ka password/email share mat karo
  • Kisi public computer pe login kiya ho toh logout zaroor karo
  • App ka link sirf team members ke saath share karo — outsiders ke saath nahi
